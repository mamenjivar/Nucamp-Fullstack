How to Install and Use VS Code's Live Server Extension

Install | Use | Video

Installation Instructions

  1. In Visual Studio Code, click the icon in the left column toolbar
  2. In the search bar that appears, type Live server
  3. Select the live server exension. It should be the top choice
  4. In the page for Live Server that appears, click the green Install button

Usage Instructions

  • Visual Studio Code must be open to a folder to use Live Server
  • In VS Code's Exploerer, right click (Windows) or Control+click (macOS) on HTML file. From the context menu that ppears, select Open with Liver Server
